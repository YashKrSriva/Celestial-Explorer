# 🌌 Celestial Explorer

<div align="center">

### 🚀 Explore the Cosmos Through NASA's Open APIs

A modern space exploration web application built with **React, TypeScript, Vite, and Tailwind CSS**, allowing users to track the International Space Station, discover NASA's Astronomy Picture of the Day, and monitor Near-Earth Objects in real time.

[🌍 Live Demo](https://celestialexplorer.netlify.app/)

</div>

---

## 📖 Overview

Celestial Explorer is an interactive astronomy dashboard that brings real-time space data directly to users through NASA's public APIs. The platform provides an immersive experience for space enthusiasts, students, and curious minds who want to explore the universe through live astronomical data and stunning imagery.

---

## ✨ Features

### 🛰️ ISS Tracking
- Track the International Space Station in real time
- View live location data
- Monitor the station's current position around Earth

### 🌠 Astronomy Picture of the Day (APOD)
- Explore NASA's daily featured astronomical image
- Read detailed scientific explanations
- Discover breathtaking views of the universe

### ☄️ Near-Earth Object Monitoring
- Monitor asteroids and other objects approaching Earth
- Access NASA's Near-Earth Object database
- View detailed information about detected objects

### 🎨 Modern User Interface
- Beautiful space-themed design
- Fully responsive across devices
- Smooth and intuitive user experience
- Interactive visual elements

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- Node.js (v18 or later)
- npm

Verify installation:

```bash
node -v
npm -v
```

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/YashKrSriva/Celestial-Explorer.git
```

Navigate to the project directory:

```bash
cd Celestial-Explorer
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```

---

## 🔑 Environment Variables

Create a `.env` file in the root directory:

```env
VITE_NASA_API_KEY=YOUR_NASA_API_KEY
```

Get a free API key from:

https://api.nasa.gov
